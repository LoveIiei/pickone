Random User Picker





Manual

English

1. Enter the user names in the text fields provided.
2. Click "Add User" to add the entered names to the user list.
3. Click "Another User" to add another input field for more names.
4. Click "Pick Random User" to randomly select a user from the list.

中文

1. 在提供的文本框中输入用户名。
2. 点击“Add User”将输入的名称添加到用户列表中。
3. 点击“Another User”添加另一个输入框以添加更多名称。
4. 点击“Pick Random User”从列表中随机选择一个用户。
